## 🚀 安装步骤

### 环境要求
- Android TV设备或模拟器
- Android 5.0 (API 21) 或更高版本
- 至少 1GB 可用存储空间
- 稳定的网络连接

### 安装方法

#### 方法1: Android Studio安装 (推荐开发者)

1. **克隆项目**
```bash
git clone <项目地址>
cd android-tv-app
```

2. **打开Android Studio**
   - 启动Android Studio
   - 选择 "Open an existing project"
   - 选择 `android-tv-app` 文件夹

3. **同步项目**
   - 等待Gradle同步完成
   - 如果提示SDK缺失，请安装Android SDK 34

4. **连接设备**
   - 使用USB线连接Android TV设备
   - 或启动Android TV模拟器

5. **运行应用**
   - 点击 "Run" 按钮
   - 选择目标设备
   - 等待安装和启动

#### 方法2: 直接安装APK (适用于用户)

1. **构建APK**
```bash
cd android-tv-app
./gradlew assembleRelease
```

2. **安装APK**
   - 将生成的APK文件传输到Android TV
   - 在TV上启用"未知来源"安装
   - 使用文件管理器安装APK

## 📱 使用方法

### 首次启动
1. 启动应用后，您将看到主界面
2. 使用遥控器方向键浏览菜单
3. 按OK键选择要访问的内容

### 主界面导航
- **首页**: 访问网站主页
- **热门视频**: 查看热门内容
- **最新更新**: 查看最新添加的内容
- **分类**: 按分类浏览内容

### 访问网站内容
1. 在主界面选择"首页"
2. 应用将打开WebView加载网站
3. 使用遥控器浏览网页内容
4. 点击视频链接开始播放

## 🎮 遥控器操作

### 基本导航
| 按键 | 功能 |

## 🔧 故障排除

### 常见问题

#### 1. 应用无法启动
**症状**: 点击应用图标无反应

**解决方案**:
- 检查设备是否满足最低系统要求
- 尝试重启设备
- 重新安装应用
- 检查存储空间是否足够

#### 2. 网站无法加载
**症状**: WebView显示空白页面或错误信息

**解决方案**:
- 检查网络连接
- 确认网站是否可访问
- 尝试切换到移动网络
- 清除应用缓存和数据

#### 3. 视频无法播放
**症状**: 点击视频后无反应或播放失败

**解决方案**:
- 确认视频URL是否有效
- 检查网络速度（建议至少5Mbps）
- 尝试其他视频文件
- 重启应用

#### 4. 遥控器无响应
**症状**: 按键后界面无反应

**解决方案**:
- 检查遥控器电池
- 确认遥控器与设备配对
- 重启应用
- 重启设备

#### 5. 应用运行缓慢
**症状**: 界面卡顿，加载缓慢

**解决方案**:
- 关闭其他正在运行的应用
- 清理设备存储空间
- 重启设备
- 降低视频质量

### 性能优化建议

1. **网络优化**
   - 使用稳定的WiFi连接
   - 考虑使用5GHz频段
   - 避免网络拥堵时段

2. **设备优化**
   - 定期重启设备
   - 清理不需要的应用
   - 更新系统到最新版本

3. **应用优化**
   - 定期清理应用缓存
   - 关闭不必要的后台功能

## ❓ 常见问题解答

### Q: 应用支持哪些视频格式？
A: 应用支持常见的视频格式，包括MP4、WebM、M3U8等。如果视频无法播放，可能是格式不兼容。

### Q: 可以自定义网站URL吗？
A: 可以。开发者可以在源代码中修改URL配置，普通用户可以通过更新版本来获取新的URL。

### Q: 应用是否需要互联网连接？
A: 是的，应用需要互联网连接来加载网站内容和视频。

### Q: 可以同时播放多个视频吗？
A: 不建议同时播放多个视频，这可能会影响设备性能和观看体验。

### Q: 如何更新应用？
A: 开发者发布新版本后，您可以下载新版本APK并覆盖安装。

### Q: 应用是否免费？
A: 是的，本应用完全免费，但使用网站内容时请遵守相关法律法规。

### Q: 如何反馈问题或建议？
