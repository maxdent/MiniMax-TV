# MiniMax TV - 快速开始指南

## 🚀 5分钟快速上手

### 第一步：准备环境
确保您的设备满足以下要求：
- ✅ Android TV 或 安卓模拟器
- ✅ Android 5.0 或更高版本
- ✅ 网络连接

### 第二步：获取应用

#### 方式一：直接安装（推荐用户）
```bash
# 1. 下载预编译的APK文件
# 2. 传输到Android TV设备
# 3. 启用"未知来源"安装
# 4. 安装APK
```

#### 方式二：从源码构建（推荐开发者）
```bash
# 1. 克隆项目
git clone <项目地址>
cd android-tv-app

# 2. 构建应用
./gradlew assembleRelease

# 3. 安装到设备
./gradlew installDebug
```

### 第三步：启动应用
1. 在Android TV主界面找到"MiniMax TV"应用图标
2. 使用遥控器点击应用图标
3. 等待应用启动

### 第四步：开始使用

#### 浏览内容
- 使用 **方向键** 在菜单间切换
- 按 **OK键** 打开内容
- 按 **BACK键** 返回上一页

#### 观看视频
1. 在主界面选择"首页"
2. 浏览网页内容
3. 点击视频开始播放
4. 使用遥控器控制播放

#### 遥控器快捷键
| 按键 | 功能 |
|------|------|
| 方向键 | 导航 |
| OK | 确认/播放/暂停 |
| BACK | 返回 |
| 左/右 | 快退/快进 |

## 📌 快速使用技巧

### 💡 导航技巧
- **快速浏览**: 长按方向键快速滚动
- **选择确认**: 点击OK键确认选择
- **返回**: 连续按BACK键快速返回主界面

### 🎬 观看视频
- **播放控制**: OK键播放/暂停
- **进度跳转**: 左右方向键快进/快退
- **全屏**: 应用会自动切换到全屏模式

### 🔍 搜索内容
1. 在主界面选择搜索功能
2. 使用遥控器输入搜索关键词
3. 浏览搜索结果
4. 选择感兴趣的内容
